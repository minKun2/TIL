# React Hooks 이란? :interrobang:
- 훅(Hooks)은 함수형 component에서도 class형 컴포넌트의 기능 또는 여러 component에서 사용할 수 없던 다양한 기능을 사용할 수 있기 해주는 기능입니다.

### Hooks 의 정점
 1. 작성할 코드가 줄어들어 가독성이 좋아진다.
 2. 정적 타입 언어로 타입 정의가 쉽다.
 3. 재사용이 가능한 로직을 쉽게 만들 수 있다.
